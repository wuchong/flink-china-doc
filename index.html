概览

本文档是针对 Apache Flink 1.1-SNAPSHOT 版本的。

Apache Flink 是一个开源的分布式流处理和批处理系统。Flink 的核心是在数据流上提供了数据分发、通信、具备容错的分布式计算。同时，Flink 在流处理引擎上构建了批处理引擎，原生支持了迭代计算、内存管理和程序优化。

第一步

  • 快速起步: 在你的本地机器上 运行一个实例 或者 编写一个实例。

  • 安装: 本地, 集群, 和 谷歌云 安装指南为你展示了如何部署 Flink。

  • 编程指南: 你可以查阅 基础概念 和 DataStream API 指南 或者 DataSet API 指南 学习如何编写第一个 Flink 程序。

  • 迁移指南: 如果你要从 Flink 0.10.x 升级到 1.0 ，请查阅 0.10 到 1.0 迁移指南。
